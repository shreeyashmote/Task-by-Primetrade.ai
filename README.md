# Trader Behavior vs Market Sentiment — Data Science Assignment

Author: Shreeyash Mote  
Folder: ds_Shreeyash_Mote

## 1. Project Overview
This project analyzes the relationship between market sentiment (Fear/Greed Index) 
and trader behavior using Hyperliquid trading data.

## 3. How to Run
1. Open notebook_1.ipynb in Google Colab.
2. Upload both datasets.
3. Run all cells.
4. Outputs are automatically saved.

## 4. Insights Summary
- Higher profitability during Greed.
- Higher leverage during Greed.
- Higher trade volume during Greed.
- Long trades dominate Greed; Short trades increase during Fear.
